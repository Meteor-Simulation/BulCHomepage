## 1. 개요

### 1.1 목적

본 문서는 BulC Homepage의 화면 구성 및 UI/UX 요소를 정의합니다.

### 1.2 화면 목록

| No | 화면 ID | 화면명 | 경로 | 인증 필요 |
|----|---------|--------|------|----------|
| 1 | SCR-001 | 메인 페이지 | `/` | X |
| 2 | SCR-002 | METEOR 페이지 | `/` (meteor 서브도메인) | X |
| 3 | SCR-003 | BUL:C 페이지 | `/` (bulc 서브도메인) | X |
| 4 | SCR-004 | VR 페이지 | `/` (vr 서브도메인) | X |
| 5 | SCR-005 | 결제 페이지 | `/payment` | O |
| 6 | SCR-006 | 결제 성공 | `/payment/success` | O |
| 7 | SCR-007 | 결제 실패 | `/payment/fail` | X |
| 8 | SCR-008 | 마이페이지 | `/mypage` | O |
| 9 | ~~SCR-009~~ | ~~관리자 페이지~~ | ~~`/admin`~~ | (삭제됨 → MyPage 통합) |
| 10 | SCR-010 | OAuth 콜백 | `/oauth/callback` | X |
| 11 | SCR-011 | OAuth 비밀번호 설정 | `/oauth/setup-password` | X |
| 12 | SCR-012 | 다운로드 페이지 | `/download` | X |
| 13 | SCR-013 | 에러 페이지 | `/error` | X |

> **참고**: 제품 페이지(METEOR, BUL:C, VR)는 서브도메인 기반 라우팅으로 전환되었습니다. `CategoryRouter`가 서브도메인을 감지하여 루트 경로(`/`)에서 해당 페이지를 표시합니다.

## 2. 공통 컴포넌트

### 2.1 Header (CMP-001)

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 컴포넌트 ID | CMP-001 |
| 컴포넌트명 | Header |
| 위치 | 모든 페이지 상단 |

#### 구성 요소

```
┌────────────────────────────────────────────────────────────────────────┐
│  [Logo]          [METEOR] [BUL:C] [VR] [More]       [로그인] [회원가입] │
└────────────────────────────────────────────────────────────────────────┘

로그인 후:
┌────────────────────────────────────────────────────────────────────────┐
│  [Logo]          [METEOR] [BUL:C] [VR] [More]       [홍길동 ▼] [로그아웃]│
└────────────────────────────────────────────────────────────────────────┘
```

#### 요소 설명

| 요소 | 타입 | 동작 |
|------|------|------|
| Logo | 이미지/링크 | 클릭 시 메인 페이지(`/`) 이동 |
| METEOR | 링크 | `/meteor` 페이지 이동 |
| BUL:C | 링크 | `/bulc` 페이지 이동 |
| VR | 링크 | `/vr` 페이지 이동 |
| More | 드롭다운 | 추가 메뉴 표시 |
| 로그인 | 버튼 | LoginModal 열기 |
| 회원가입 | 버튼 | SignupModal 열기 |
| 사용자명 | 드롭다운 | 마이페이지, 관리자 페이지 링크 |
| 로그아웃 | 버튼 | 로그아웃 처리 |

#### 상태별 표시

| 상태 | 표시 요소 |
|------|----------|
| 비로그인 | 로그인, 회원가입 버튼 |
| 일반 사용자 | 사용자명, 마이페이지 링크 |
| 관리자 | 사용자명, 마이페이지, 관리자 페이지 링크 |

#### 화면 구성

스크롤 기반 섹션 네비게이션 구조로, 상단 서브 네비게이션 바를 통해 각 섹션으로 이동할 수 있습니다.
`useScrollNav` 커스텀 훅을 사용하며, 스크롤 위치에 따라 활성 섹션이 자동 하이라이트됩니다.

```
┌────────────────────────────────────────────────────────────────────────┐
│                              [Header]                                   │
│  [Intro] [Comparison] [Core Values] [Workflow] [Report] [Get Started]  │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  ┌─ #hero ──────────────────────────────────────────────────────────┐ │
│  │  Hero Section                                                     │ │
│  │  - BUL:C 제품명 및 슬로건                                         │ │
│  │  - CTA 버튼 (구매하기)                                            │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #comparison ────────────────────────────────────────────────────┐ │
│  │  Comparison Section                                               │ │
│  │  - 기존 방식 vs BUL:C 비교                                        │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #core-values ───────────────────────────────────────────────────┐ │
│  │  Core Values Section                                              │ │
│  │  - 핵심 가치 카드 (lucide-react 아이콘)                           │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #workflow ──────────────────────────────────────────────────────┐ │
│  │  Workflow Section                                                 │ │
│  │  - 사용 절차 안내                                                  │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #report ────────────────────────────────────────────────────────┐ │
│  │  Report Section                                                   │ │
```

```css
/* Primary Button */

border: 1px solid #333;
```

