## 1. 개요

### 1.1 목적

본 문서는 BulC Homepage의 화면 구성 및 UI/UX 요소를 정의합니다.

### 1.2 화면 목록

| No | 화면 ID | 화면명 | 경로 | 인증 필요 |

| 1 | SCR-001 | 메인 페이지 | `/` | X |
| 2 | SCR-002 | METEOR 페이지 | `/meteor` | X |
| 3 | SCR-003 | BUL:C 페이지 | `/bulc` | X |
| 4 | SCR-004 | VR 페이지 | `/vr` | X |

### 2.1 Header (CMP-001)

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 컴포넌트 ID | CMP-001 |
| 컴포넌트명 | Header |
| 위치 | 모든 페이지 상단 |

#### 구성 요소

```
┌────────────────────────────────────────────────────────────────────────┐
│  [Logo]          [METEOR] [BUL:C] [VR] [More]       [로그인] [회원가입] │
└────────────────────────────────────────────────────────────────────────┘

로그인 후:
┌────────────────────────────────────────────────────────────────────────┐
│  [Logo]          [METEOR] [BUL:C] [VR] [More]       [홍길동 ▼] [로그아웃]│
└────────────────────────────────────────────────────────────────────────┘
```

#### 요소 설명

| 요소 | 타입 | 동작 |
|------|------|------|
| Logo | 이미지/링크 | 클릭 시 메인 페이지(`/`) 이동 |
| METEOR | 링크 | `/meteor` 페이지 이동 |
| BUL:C | 링크 | `/bulc` 페이지 이동 |
| VR | 링크 | `/vr` 페이지 이동 |
| More | 드롭다운 | 추가 메뉴 표시 |
| 로그인 | 버튼 | LoginModal 열기 |
| 회원가입 | 버튼 | SignupModal 열기 |
| 사용자명 | 드롭다운 | 마이페이지, 관리자 페이지 링크 |
| 로그아웃 | 버튼 | 로그아웃 처리 |

#### 상태별 표시

| 상태 | 표시 요소 |
|------|----------|
| 비로그인 | 로그인, 회원가입 버튼 |
| 일반 사용자 | 사용자명, 마이페이지 링크 |
| 관리자 | 사용자명, 마이페이지, 관리자 페이지 링크 |

#### 결제 흐름

```
1. 상품 선택
   ↓
2. 요금제 선택
   ↓
3. 결제자 정보 입력 (이름, 이메일, 전화번호)
   ↓
4. 쿠폰 코드 적용 (선택)
   ↓
5. [결제하기] 버튼 클릭
   ↓
6. 토스페이먼츠 결제창
   ↓
7. 결제 수단 선택 (카드, 계좌이체, 간편결제)
   ↓
8. 결제 완료 → /payment/success
   (결제 실패 → /payment/fail)
```
