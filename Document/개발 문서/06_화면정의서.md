
#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-003 |
| 화면명 | BUL:C 페이지 |
| 경로 | `/bulc` |
| 파일 | `CategoryPages/BULC/BulC.tsx` |

#### 화면 구성

스크롤 기반 섹션 네비게이션 구조로, 상단 서브 네비게이션 바를 통해 각 섹션으로 이동할 수 있습니다.
`useScrollNav` 커스텀 훅을 사용하며, 스크롤 위치에 따라 활성 섹션이 자동 하이라이트됩니다.

```
┌────────────────────────────────────────────────────────────────────────┐
│                              [Header]                                   │
│  [Intro] [Comparison] [Core Values] [Workflow] [Report] [Get Started]  │
├────────────────────────────────────────────────────────────────────────┤
│                                                                        │
│  ┌─ #hero ──────────────────────────────────────────────────────────┐ │
│  │  Hero Section                                                     │ │
│  │  - BUL:C 제품명 및 슬로건                                         │ │
│  │  - CTA 버튼 (구매하기)                                            │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #comparison ────────────────────────────────────────────────────┐ │
│  │  Comparison Section                                               │ │
│  │  - 기존 방식 vs BUL:C 비교                                        │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #core-values ───────────────────────────────────────────────────┐ │
│  │  Core Values Section                                              │ │
│  │  - 핵심 가치 카드 (lucide-react 아이콘)                           │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #workflow ──────────────────────────────────────────────────────┐ │
│  │  Workflow Section                                                 │ │
│  │  - 사용 절차 안내                                                  │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #report ────────────────────────────────────────────────────────┐ │
│  │  Report Section                                                   │ │
│  │  - ASET/RSET 보고서 예시                                          │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
│  ┌─ #cta ───────────────────────────────────────────────────────────┐ │
│  │  CTA Section                                                      │ │
│  │  - 최종 구매 유도                                                  │ │
│  └──────────────────────────────────────────────────────────────────┘ │
│                                                                        │
├────────────────────────────────────────────────────────────────────────┤
│                              [Footer]                                   │
└────────────────────────────────────────────────────────────────────────┘
```

#### 서브 네비게이션

| 항목 | ID | 라벨 | 섹션 컴포넌트 |
|------|-----|------|-------------|
| 1 | hero | Intro | HeroSection |
| 2 | comparison | Comparison | ComparisonSection |
| 3 | core-values | Core Values | CoreValuesSection |
| 4 | workflow | Workflow | WorkflowSection |
| 5 | report | Report | ReportSection |
| 6 | cta | Get Started | CTASection |

#### 동작 설명

| 요소 | 동작 |
|------|------|
| 서브 네비게이션 항목 | 클릭 시 해당 섹션으로 스크롤 이동 |
| 스크롤 | 현재 보이는 섹션에 맞춰 서브 네비게이션 활성 항목 자동 변경 |
| 구매하기 버튼 | 로그인 시 `/payment` 이동, 비로그인 시 LoginModal 표시 |
| 로고 | 서브도메인 접속 시 Hero 섹션으로 이동, 일반 접속 시 `/` 이동 |
