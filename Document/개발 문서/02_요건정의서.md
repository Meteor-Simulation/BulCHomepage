## 1. 개요

### 1.2 범위

- 웹 프론트엔드 기능 요구사항

### 2.1 사용자 인증 (AUTH)

#### AUTH-001: 회원가입

| 항목 | 내용 |
|------|------|
| **요구사항 ID** | AUTH-001 |
| **요구사항명** | 이메일 기반 회원가입 |
| **우선순위** | 필수 |
| **설명** | 사용자가 이메일과 비밀번호로 계정을 생성할 수 있어야 한다 |

**상세 요구사항:**

| No | 상세 내용 | 비고 |
|----|----------|------|

| 5 | 필수 정보 입력 | 이름, 전화번호 |
| 6 | 약관 동의 확인 | 서비스 이용약관, 개인정보처리방침 |
