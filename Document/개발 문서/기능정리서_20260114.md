## 1. 홈페이지 제작

### a. 인증 기능

   i. 회원가입 (이메일 인증 + 비밀번호 설정)

   ii. 이메일 인증 코드 발송 (6자리, 10분 유효)

   iii. 이메일 중복 확인 (실시간 검증)

   iv. 로그인 (이메일/비밀번호 JWT 인증)

   v. 로그아웃 (토큰 무효화, Token Blacklist)

   vi. 토큰 갱신 (Access Token 1시간, Refresh Token 24시간)

   vii. 비밀번호 재설정 (이메일 인증 후 변경)

   viii. 소셜 로그인 - Google OAuth2

   ix. 소셜 로그인 - Naver OAuth2

   x. 소셜 로그인 - Kakao OAuth2

   xi. OAuth 신규 사용자 비밀번호 설정

### b. 사용자 관리 기능

   i. 내 정보 조회 (이메일, 이름, 전화번호, 연동 계정)

   ii. 내 정보 수정 (이름, 전화번호 변경)

   iii. 비밀번호 변경 (현재 비밀번호 확인 필수)

   iv. 계정 비활성화 (회원 탈퇴)

   v. 계정 재활성화 (탈퇴 계정 복구)

### c. 상품/요금제 기능

   i. 상품 목록 조회 (METEOR, BUL:C, VR 등)

   ii. 상품별 요금제 조회 (Basic, Pro, Enterprise)

   iii. 통화별 가격 표시 (KRW, USD)

### d. 결제 기능

   i. 토스페이먼츠 결제 처리

   ii. 카드 결제 (신용/체크카드)

   iii. 계좌이체 (실시간 이체)

   iv. 가상계좌 (무통장입금)

   v. 간편결제 (토스페이, 카카오페이 등)

   vi. 쿠폰/프로모션 코드 적용

   vii. 결제 금액 서버측 검증

   viii. 결제 성공/실패 처리

### e. 구독 관리 기능

   i. 내 구독 목록 조회

   ii. 구독 상세 정보 조회 (시작일, 종료일, 상태)

   iii. 자동 갱신 활성화

   iv. 자동 갱신 비활성화

   v. 구독별 결제 이력 조회

   vi. 빌링키 발급 (자동결제 카드 등록)

   vii. 빌링키 목록 조회 (등록된 카드 목록)

   viii. 기본 결제 수단 변경

   ix. 빌링키 삭제 (카드 삭제)

### f. 라이선스 관리 기능

   i. 내 라이선스 목록 조회

   ii. 라이선스 검증 및 기기 활성화

   iii. Heartbeat 세션 유지 (15분 주기)

   iv. 강제 활성화 (기존 세션 종료 후 새 기기 활성화)

   v. 기기 비활성화

   vi. 라이선스 상세 조회 (상태, 만료일, 활성 기기)

   vii. 오프라인 토큰 발급 (로컬 검증용)

   viii. 동시 접속 제한 (maxActivations 기준)

   ix. Stale 세션 자동 정리 (Auto-Resolve)

### g. 관리자 기능 - 사용자 관리

   i. 사용자 목록 조회

   ii. 사용자 권한 변경 (Admin, Manager, User)

### h. 관리자 기능 - 상품 관리

   i. 전체 상품 목록 조회 (비활성 포함)

   ii. 상품 생성

   iii. 상품 수정

   iv. 상품 활성화/비활성화

### i. 관리자 기능 - 요금제 관리

   i. 요금제 목록 조회 (상품별 필터링)

   ii. 요금제 생성

   iii. 요금제 수정

   iv. 요금제 활성화/비활성화

   v. 라이선스 플랜 연계

### j. 관리자 기능 - 라이선스 관리

   i. 전체 라이선스 목록 조회 (필터/검색)

   ii. 라이선스 상세 조회 (활성화 기기 포함)

   iii. 라이선스 활성화 (PENDING → ACTIVE)

   iv. 라이선스 정지 (SUSPENDED)

   v. 라이선스 플랜 생성

   vi. 라이선스 플랜 수정

   vii. 라이선스 플랜 삭제 (Soft Delete)

### k. 관리자 기능 - 결제 조회

   i. 전체 결제 내역 조회

   ii. 기간별 필터링

   iii. 상태별 필터링 (완료, 실패, 환불)

### l. 기타 기능

   i. 문의하기 (MS Graph API 이메일 발송)

   ii. 헬스 체크 (서버 상태 확인)

### m. 화면 구성

   i. 메인 페이지

   ii. METEOR 제품 페이지

   iii. BUL:C 제품 페이지

   iv. VR 제품 페이지

   v. 결제 페이지

   vi. 결제 성공/실패 페이지

   vii. 마이페이지 (기본정보, 구독, 라이선스, 결제수단, 보안 탭)

   viii. 관리자 페이지 (대시보드, 사용자, 상품, 요금제, 라이선스, 결제 메뉴)

   ix. OAuth 콜백 페이지

   x. OAuth 비밀번호 설정 페이지

### n. 권한 체계

   i. Admin (000) - 전체 관리 권한

   ii. Manager (001) - 일부 관리 기능

   iii. User (002) - 일반 사용자
